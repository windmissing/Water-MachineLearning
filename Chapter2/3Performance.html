1.2. 2.3.2 查准率、查全率与F1

查准率 = precision = 准确率：挑出的正例中有多少真的是正例
查全率 = recall = 召回率：多少正例被挑出来了

1.2.1. 混淆矩阵

横：预测
纵：真实		 P N
P TP FN
N FP TN

T：真实=预测
F：真实！=预测
P：正例
N：负例

查准率P=TPTP+FPP = \frac{TP}{TP+FP}P=TP+FPTP​
查全率R=TPTP+FNR = \frac{TP}{TP+FN}R=TP+FNTP​

对同一个模型来说，P和R是矛盾的度量

1.2.2. P-R曲线


根据P-R曲线比较两个模型的好坏：

  1. 若曲线不交叉（例如B和C），则外侧曲线的对应的模型B优于内侧曲线对应的模型
  2. 若曲线交叉（例如A和B），则只能在具体的条件下比较
  3. 比较曲线下面的面积，大则优
  4. 比较平衡点（Break-Even Point）的值，P=R时的值，大则优
  5. F1度量，调和平均
    F1=2×P×RP+R=2×TP样本数+TP−TN F1 = \frac{2 \times P \times R}{P + R} = \frac{2 \times TP}{\text{样本数} + TP - TN} F1=​P+R​​2×P×R​​=​样本数+TP−TN​​2×TP​​
  6. FβF_\betaFβ​度量，加权调和平均
    Fβ=(1+β2)×P×R(β2×P)+R F_\beta = \frac{(1+\beta^2)\times P \times R}{(\beta^2 \times P) + R} F​β​​=​(β​2​​×P)+R​​(1+β​2​​)×P×R​​

β<1\beta < 1β<1：更注重查准率
β=1\beta = 1β=1：相当于F1
β>1\beta > 1β>1：更注重查全率
